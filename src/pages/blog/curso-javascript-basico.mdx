## Operadores Aritméticos

> > Los operadores aritméticos en JavaScript se utilizan para realizar cálculos matemáticos básicos. A continuación se muestran los operadores aritméticos disponibles en JavaScript:

- **(+)** (suma): se utiliza para sumar dos valores.
- **(-)** (resta): se utiliza para restar un valor de otro.
- **(\*)** (multiplicación): se utiliza para multiplicar dos valores.
- **(/)** (división): se utiliza para dividir un valor por otro.
- **(%)** (módulo): se utiliza para obtener el resto de la división de dos valores.
- **(++)** (incremento): se utiliza para aumentar el valor de una variable en uno.
- **(--)** (decremento): se utiliza para disminuir el valor de una variable en uno.

> ## Operadores Lógicos

Los operadores lógicos en JavaScript se utilizan para realizar operaciones lógicas. A continuación se muestran los operadores lógicos disponibles en JavaScript:

- **(&&)** (AND lógico): devuelve **true** si ambos operandos son verdaderos.
- **(||)** (OR lógico): devuelve **true** si al menos uno de los operandos es verdadero.
- **!** (NOT lógico): devuelve el valor opuesto del operando.

> ## Operadores de Comparación

Los operadores de comparación en JavaScript se utilizan para comparar dos valores. A continuación se muestran los operadores de comparación disponibles en JavaScript:

- `==` (igual a): devuelve **true** si ambos operandos son iguales.
- `!=` (diferente de): devuelve **true** si ambos operandos no son iguales.
- `>` (mayor que): devuelve **true** si el primer operando es mayor que el segundo.
- `<` (menor que): devuelve **true** si el primer operando es menor que el segundo.
- `>=` (mayor o igual que): devuelve **true** si el primer operando es mayor o igual que el segundo.
- `<=` (menor o igual que): devuelve **true** si el primer operando es menor o igual que el segundo.
- `===` (igualdad estricta): devuelve **true** si ambos operandos son iguales y del mismo tipo de dato.
- `!==` (diferencia estricta): devuelve **true** si ambos operandos no son iguales o no son del mismo tipo de dato.

> ## Operadores de Asignación

Los operadores de asignación en JavaScript se utilizan para asignar un valor a una variable. A continuación se muestran los operadores de asignación disponibles en JavaScript:

- `=` (asignación): se utiliza para asignar un valor a una variable.
- `+=` (suma y asignación): se utiliza para sumar un valor a una variable y asignar el resultado a la misma variable.
- `-=` (resta y asignación): se utiliza para restar un valor a una variable y asignar el resultado a la misma variable.
- `*=` (multiplicación y asignación): se utiliza para multiplicar un valor a una variable y asignar el resultado a la misma variable.
- `/=` (división y asignación): se utiliza para dividir un valor a una variable y asignar el resultado a la misma variable.
- `%=` (módulo y asignación): se utiliza para obtener el resto de la división de dos valores y asignar el resultado a la misma variable.
- `<<=` (desplazamiento a la izquierda y asignación): se utiliza para desplazar los bits de un valor a la izquierda y asignar el resultado a la misma variable.
- `>>=` (desplazamiento a la derecha y asignación): se utiliza para desplazar los bits de un valor a la derecha y asignar el resultado a la misma variable.
- `>>>=` (desplazamiento a la derecha sin signo y asignación): se utiliza para desplazar los bits de un valor a la derecha sin signo y asignar el resultado a la misma variable.
- `&=` (AND a nivel de bits y asignación): se utiliza para realizar una operación AND a nivel de bits entre dos valores y asignar el resultado a la misma variable.
- `^=` (OR exclusivo a nivel de bits y asignación): se utiliza para realizar una operación OR exclusivo a nivel de bits entre dos valores y asignar el resultado a la misma variable.
- `|=` (OR a nivel de bits y asignación): se utiliza para realizar una operación OR a nivel de bits entre dos valores y asignar el resultado a la misma variable.

## Condicionales en JavaScript

Los condicionales en JavaScript se utilizan para realizar diferentes acciones en función de si se cumple o no una determinada condición. A continuación se muestran los diferentes tipos de condicionales disponibles en JavaScript:

## if

La sentencia if se utiliza para realizar una acción si se cumple una determinada condición. La sintaxis es la siguiente:

```jsx
if (condicion) {
  // código a ejecutar si la condición se cumple
}
```

Por ejemplo:

```jsx
let edad = 18;

if (edad >= 18) {
  console.log("Eres mayor de edad");
}
```

En este caso, si la variable edad es mayor o igual a 18, se mostrará por consola el mensaje "Eres mayor de edad".

## if...else

La sentencia if...else se utiliza para realizar una acción si se cumple una determinada condición, y otra acción si no se cumple. La sintaxis es la siguiente:

```jsx
if (condicion) {
  // código a ejecutar si la condición se cumple
} else {
  // código a ejecutar si la condición no se cumple
}
```

Por ejemplo:

```jsx
let edad = 16;

if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}
```

En este caso, si la variable edad es mayor o igual a 18, se mostrará por consola el mensaje "Eres mayor de edad"; de lo contrario, se mostrará el mensaje "Eres menor de edad".

## if... else if...else

La sentencia if...else if...else se utiliza para realizar una acción si se cumple una determinada condición, otra acción si se cumple otra condición, y una acción por defecto si no se cumple ninguna de las condiciones anteriores. La sintaxis es la siguiente:

```jsx
if (condicion1) {
  // código a ejecutar si la condición 1 se cumple
} else if (condicion2) {
  // código a ejecutar si la condición 2 se cumple
} else {
  // código a ejecutar si ninguna de las condiciones se cumple
}
```

Por ejemplo:

```jsx
let nota = 7;

if (nota >= 9) {
  console.log("Sobresaliente");
} else if (nota >= 7) {
  console.log("Notable");
} else if (nota >= 5) {
  console.log("Aprobado");
} else {
  console.log("Suspenso");
}
```

En este caso, si la variable nota es mayor o igual a 9, se mostrará por consola el mensaje "Sobresaliente"; si la variable `nota` es mayor o igual a `7` pero menor que `9`, se mostrará el mensaje "Notable"; si la variable `nota` es mayor o igual a `5` pero menor que `7`, se mostrará el mensaje "Aprobado"; y si la variable `nota` es menor que `5`, se mostrará el mensaje "Suspenso".

¡Ahora ya sabes cómo utilizar los diferentes tipos de condicionales en JavaScript!

## Arrays en JavaScript

Un array en JavaScript es una estructura de datos que nos permite almacenar una colección de elementos en una sola variable. Cada elemento en un array tiene una posición numérica llamada índice, que comienza en 0.

## Crear un array

Podemos crear un array en JavaScript utilizando la siguiente sintaxis:

```jsx
let miArray = [elemento1, elemento2, elemento3];
```

También podemos crear un array vacío utilizando la siguiente sintaxis:

```jsx
let miArray = [];
```

## Acceder a los elementos de un array

Podemos acceder a los elementos de un array utilizando su índice. La sintaxis es la siguiente:

```jsx
let miArray = [elemento1, elemento2, elemento3];
console.log(miArray[0]); // elemento1
```

## Modificar los elementos de un array

Podemos modificar los elementos de un array utilizando su índice. La sintaxis es la siguiente:

```jsx
let miArray = [elemento1, elemento2, elemento3];
miArray[0] = nuevoElemento;
console.log(miArray); // [nuevoElemento, elemento2, elemento3]
```

## Añadir elementos a un array

Podemos añadir elementos a un array utilizando el método push(). La sintaxis es la siguiente:

```jsx
let miArray = [elemento1, elemento2, elemento3];
miArray.push(nuevoElemento);
console.log(miArray); // [elemento1, elemento2, elemento3, nuevoElemento]
```

## Eliminar elementos de un array

Podemos eliminar elementos de un array utilizando el método splice(). La sintaxis es la siguiente:

```jsx
let miArray = [elemento1, elemento2, elemento3];
miArray.splice(1, 1);
console.log(miArray); // [elemento1, elemento3]
```

En este ejemplo, el primer parámetro del método splice() indica el índice del primer elemento que queremos eliminar, y el segundo parámetro indica la cantidad de elementos que queremos eliminar a partir del índice especificado.

## Iterar sobre un array

Podemos iterar sobre los elementos de un array utilizando un bucle for. La sintaxis es la siguiente:

```jsx
let miArray = [elemento1, elemento2, elemento3];
for (let i = 0; i < miArray.length; i++) {
  console.log(miArray[i]);
}
```

## Métodos útiles en arrays

JavaScript también proporciona una gran cantidad de métodos útiles que podemos utilizar en los arrays, como sort(), `reverse()`, `slice()`, `join()`, `map()`, `filter()`, `reduce()`, entre otros. Estos métodos nos permiten realizar operaciones comunes en los arrays de una manera más sencilla y eficiente.

¡Eso es todo! Ahora ya sabes cómo trabajar con arrays en JavaScript.

## Bucles e Iteración en JavaScript

### Introducción a los Bucles

Los bucles son estructuras fundamentales en la programación que permiten ejecutar un bloque de código repetidamente hasta que se cumpla una condición específica.

## Tipos de Bucles

### 1. Bucle "for"

El bucle "for" se utiliza para ejecutar un bloque de código un número específico de veces.

```jsx
for (let i = 0; i < 5; i++) {
  console.log("Iteración #" + i);
}
```

### 2. Bucle "while"

El bucle "while" se ejecuta mientras una condición dada sea verdadera.

```jsx
let contador = 0;
while (contador < 3) {
  console.log("Contador: " + contador);
  contador++;
}
```

### 3. Bucle "do-while"

Similar al bucle "while", el bucle "do-while" ejecuta el bloque de código al menos una vez y luego continúa mientras la condición sea verdadera.

```jsx
let x = 0;
do {
  console.log("x es igual a " + x);
  x++;
} while (x < 3);
```

## Uso de Bucles

### Iteración a través de un Array

Los bucles son útiles para recorrer elementos de un array.

```jsx
const frutas = ["manzana", "plátano", "naranja"];
for (let i = 0; i < frutas.length; i++) {
  console.log("Fruta: " + frutas[i]);
}
```

### Bucle "for...of"

El bucle "for...of" permite iterar sobre valores de un objeto iterable, como un array.

```jsx
const numeros = [1, 2, 3, 4, 5];
for (const numero of numeros) {
  console.log("Número: " + numero);
}
```

## Control de Bucles

### Instrucción "break"

La instrucción "break" se utiliza para salir de un bucle antes de que se cumpla la condición.

```jsx
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log("Valor: " + i);
}
```

### Instrucción "continue"

La instrucción "continue" se utiliza para omitir la iteración actual y continuar con la siguiente.

```jsx
for (let i = 0; i < 5; i++) {
  if (i === 2) {
    continue;
  }
  console.log("Iteración: " + i);
}
```

## Funciones en JavaScript

### Introducción a las Funciones

Las funciones son bloques de código reutilizables que pueden ser llamados con un nombre y opcionalmente recibir argumentos para realizar tareas específicas.

### Declaración de Funciones

```jsx
function saludar(nombre) {
  console.log("¡Hola, " + nombre + "!");
}
```

### Llamada de Funciones

```jsx
saludar("Juan"); // Imprime: ¡Hola, Juan!
```

### Parámetros y Argumentos

Las funciones pueden recibir parámetros (variables) que actúan como marcadores de posición para los valores que se les pasan al llamar la función.

```jsx
function sumar(a, b) {
  return a + b;
}

const resultado = sumar(3, 5); // resultado es 8
```

### Valores de Retorno

Las funciones pueden devolver valores utilizando la palabra clave **return**.

```jsx
function multiplicar(x, y) {
  return x * y;
}

const producto = multiplicar(4, 6); // producto es 24
```

### Funciones Anónimas

Las funciones anónimas no tienen nombre y se asignan a variables o se pasan como argumentos a otras funciones.

```jsx
const saludar = function (nombre) {
  console.log("¡Hola, " + nombre + "!");
};
```

### Funciones de Flecha (Arrow Functions)

Las funciones de flecha son una sintaxis más concisa para definir funciones.

```jsx
const restar = (a, b) => a - b;
```

### Ámbito de las Variables

Las variables declaradas dentro de una función tienen un ámbito local y solo son visibles dentro de esa función.

```jsx
function imprimirMensaje() {
  const mensaje = "Hola desde la función";
  console.log(mensaje);
}

imprimirMensaje(); // Imprime: Hola desde la función
console.log(mensaje); // Generará un error
```

### Funciones como Parámetros

Las funciones pueden ser pasadas como argumentos a otras funciones.

```jsx
function ejecutarOperacion(func, a, b) {
  return func(a, b);
}

const resultado = ejecutarOperacion(sumar, 10, 5); // resultado es 15
```

### Cierre (Closures)

Los closures permiten que una función acceda a variables de su ámbito externo incluso después de que esa función haya terminado de ejecutarse.

```jsx
function contador() {
  let contador = 0;
  return function () {
    contador++;
    console.log(contador);
  };
}

const incrementar = contador();
incrementar(); // Imprime: 1
incrementar(); // Imprime: 2
```

## Programación Orientada a Objetos en JavaScript

### Introducción a la Programación Orientada a Objetos (POO)

La Programación Orientada a Objetos es un paradigma de programación que se basa en el concepto de objetos. Un objeto es una entidad que contiene datos y métodos, y se utiliza para modelar conceptos del mundo real de manera más eficiente en el código.

## Clases y Objetos

### Clases

```jsx
class Persona {
  constructor(nombre, edad) {
    this.nombre = nombre;
    this.edad = edad;
  }

  saludar() {
    console.log(`Hola, soy ${this.nombre} y tengo ${this.edad} años.`);
  }
}
```

### Creación de Objetos

Los objetos son instancias de una clase. Se crean utilizando el constructor de la clase.

```jsx
const persona1 = new Persona("Juan", 30);
const persona2 = new Persona("María", 25);

persona1.saludar(); // Imprime: Hola, soy Juan y tengo 30 años.
persona2.saludar(); // Imprime: Hola, soy María y tengo 25 años.
```

## Propiedades y Métodos

### Propiedades

Las propiedades son variables que pertenecen a un objeto y almacenan sus datos.

```jsx
class Rectangulo {
  constructor(ancho, alto) {
    this.ancho = ancho;
    this.alto = alto;
  }

  calcularArea() {
    return this.ancho * this.alto;
  }
}

const rectangulo = new Rectangulo(10, 5);
console.log("Área:", rectangulo.calcularArea()); // Imprime: Área: 50
```

### Métodos

Los métodos son funciones asociadas a un objeto que realizan operaciones sobre sus datos.

```jsx
class Coche {
  constructor(marca, modelo) {
    this.marca = marca;
    this.modelo = modelo;
    this.encendido = false;
  }

  encender() {
    this.encendido = true;
    console.log("El coche está encendido.");
  }

  apagar() {
    this.encendido = false;
    console.log("El coche está apagado.");
  }
}

const miCoche = new Coche("Toyota", "Corolla");
miCoche.encender(); // Imprime: El coche está encendido.
miCoche.apagar(); // Imprime: El coche está apagado.
```

## Herencia y Polimorfismo

### Herencia

La herencia permite que una clase (subclase) herede propiedades y métodos de otra clase (superclase).

```jsx
class Animal {
  constructor(nombre) {
    this.nombre = nombre;
  }

  sonido() {
    console.log("Hace un sonido indefinido.");
  }
}

class Perro extends Animal {
  sonido() {
    console.log("Hace un ladrido.");
  }
}

const miPerro = new Perro("Buddy");
miPerro.sonido(); // Imprime: Hace un ladrido.
```

### Polimorfismo

El polimorfismo permite que objetos de diferentes clases respondan a la misma interfaz de manera diferente.

```jsx
class Figura {
  calcularArea() {
    console.log("Área indefinida.");
  }
}

class Circulo extends Figura {
  constructor(radio) {
    super();
    this.radio = radio;
  }

  calcularArea() {
    return Math.PI * this.radio * this.radio;
  }
}

class Cuadrado extends Figura {
  constructor(lado) {
    super();
    this.lado = lado;
  }

  calcularArea() {
    return this.lado * this.lado;
  }
}

const circulo = new Circulo(5);
const cuadrado = new Cuadrado(4);
console.log("Área del círculo:", circulo.calcularArea());
// Imprime: Área del círculo: 78.53981633974483
console.log("Área del cuadrado:", cuadrado.calcularArea());
// Imprime: Área del cuadrado: 16
```

## Métodos de cadenas en JavaScript

### 1. concat()

Concatena dos o más cadenas y devuelve una nueva cadena.

Ejemplo:

```jsx
let str1 = "Hola";
let str2 = "Mundo";
let str3 = str1.concat(" ", str2);
console.log(str3); // "Hola Mundo"
```

### 2. slice()

Extrae una sección de una cadena y devuelve una nueva cadena.

Ejemplo:

```jsx
let str = "Hola Mundo";
let subStr = str.slice(0, 4);
console.log(subStr); // "Hola"
```

### 3. substr()

Extrae una sección de una cadena comenzando desde el índice especificado y devuelve una nueva cadena.

Ejemplo:

```jsx
let str = "Hola Mundo";
let subStr = str.substr(5, 5);
console.log(subStr); // "Mundo"
```

### 4. toLowerCase()

Convierte una cadena a minúsculas y devuelve una nueva cadena.

Ejemplo:

```jsx
let str = "HOLA MUNDO";
let lowerStr = str.toLowerCase();
console.log(lowerStr); // "hola mundo"
```

### 5. toUpperCase()

Convierte una cadena a mayúsculas y devuelve una nueva cadena.

Ejemplo:

```jsx
let str = "hola mundo";
let upperStr = str.toUpperCase();
console.log(upperStr); // "HOLA MUNDO"
```

### 6. trim()

Elimina los espacios en blanco al principio y al final de una cadena y devuelve una nueva cadena.

Ejemplo:

```jsx
let str = "    Hola Mundo    ";
let trimmedStr = str.trim();
console.log(trimmedStr); // "Hola Mundo"
```

### 7. replace()

Reemplaza una parte de una cadena con otra y devuelve una nueva cadena.

Ejemplo:

```jsx
let str = "Hola Mundo";
let newStr = str.replace("Mundo", "Amigo");
console.log(newStr); // "Hola Amigo"
```

### 8. indexOf()

Devuelve el índice de la primera aparición de una subcadena en una cadena.

Ejemplo:

```jsx
let str = "Hola Mundo";
let index = str.indexOf("Mundo");
console.log(index); // 5
```

### 9. lastIndexOf()

Devuelve el índice de la última aparición de una subcadena en una cadena.

Ejemplo:

```jsx
let str = "Hola Mundo, Hola Amigo";
let index = str.lastIndexOf("Hola");
console.log(index); // 12
```

### 10. charAt()

Devuelve el carácter en el índice especificado de una cadena.

Ejemplo:

```jsx
let str = "Hola Mundo";
let char = str.charAt(0);
console.log(char); // "H"
```

### 11. split()

Divide una cadena en un array de subcadenas y devuelve el array.

Ejemplo:

```jsx
let str = "Hola,Mundo,Amigo";
let arr = str.split(",");
console.log(arr); // ["Hola", "Mundo", "Amigo"]
```

## Métodos de arrays en JavaScript

### 1. concat()

Combina dos o más arrays y devuelve un nuevo array.

Ejemplo:

```jsx
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let arr3 = arr1.concat(arr2);
console.log(arr3); // [1, 2, 3, 4, 5, 6]
```

### 2. push()

Agrega uno o más elementos al final de un array.

Ejemplo:

```jsx
let arr = [1, 2, 3];
arr.push(4);
console.log(arr); // [1, 2, 3, 4]
```

### 3. pop()

Elimina el último elemento de un array y lo devuelve.

Ejemplo:

```jsx
let arr = [1, 2, 3];
let lastElement = arr.pop();
console.log(lastElement); // 3
console.log(arr); // [1, 2]
```

### 4. shift()

Elimina el primer elemento de un array y lo devuelve.

Ejemplo:

```jsx
let arr = [1, 2, 3];
let firstElement = arr.shift();
console.log(firstElement); // 1
console.log(arr); // [2, 3]
```

### 5. unshift()

Agrega uno o más elementos al inicio de un array.

Ejemplo:

```jsx
let arr = [1, 2, 3];
arr.unshift(0);
console.log(arr); // [0, 1, 2, 3]
```

### 6. slice()

Devuelve una sección de un array como un nuevo array.

Ejemplo:

```jsx
let arr = [1, 2, 3, 4, 5];
let slicedArr = arr.slice(1, 4);
console.log(slicedArr); // [2, 3, 4]
```

### 7. splice()

Agrega y/o elimina elementos de un array.

Ejemplo:

```jsx
let arr = [1, 2, 3, 4, 5];
```

### 8. reverse()

### 9. join()

Une todos los elementos de un array en una cadena y devuelve la cadena.


### 10. sort()

Ordena los elementos de un array.


### 11. filter()

Crea un nuevo array con todos los elementos que pasan la prueba implementada por la función proporcionada.

### 12. map()

Crea un nuevo array con los resultados de la llamada a la función proporcionada en cada elemento del array original.

Ejemplo:

Aplica una función acumulativa a cada elemento de un array y devuelve un único valor resultante.

### 14. forEach()

La propiedad textContent nos permite modificar el contenido de texto de un elemento:

```jsx
const elemento = document.querySelector("#mi-elemento");
```

```jsx
const contenedor = document.querySelector("#mi-contenedor");
```

## Obtención y Modificación de Childs en JavaScript

Los nodos hijos, también conocidos como "childs", son los elementos o nodos que están directamente contenidos dentro de otro elemento en el DOM. Puedes acceder y manipular los hijos de un elemento utilizando varias propiedades y métodos.

## Obtener Nodos Hijos

### childNodes

La propiedad childNodes devuelve una lista de todos los nodos hijos de un elemento, incluyendo nodos de texto y comentarios.

```jsx
const contenedor = document.getElementById("mi-contenedor");
```

La propiedad children devuelve una lista de todos los elementos hijos de un elemento, excluyendo nodos de texto y comentarios.

```jsx
const contenedor = document.getElementById("mi-contenedor");
const elementosHijos = contenedor.children;
```

Las propiedades firstChild y lastChild devuelven el primer y último nodo hijo, respectivamente.

```jsx

const primerHijo = contenedor.firstChild;
const ultimoHijo = contenedor.lastChild;
```

El método **removeChild()** elimina un nodo hijo específico de un elemento.

```jsx
const contenedor = document.getElementById("mi-contenedor");
```

El método **replaceChild()** reemplaza un nodo hijo con otro nuevo.

## Métodos de Childs en JavaScript

## Introducción a los Métodos de Childs

Los métodos que te presentaré a continuación son útiles para manipular y trabajar con los nodos hijos de un elemento en el DOM (Document Object Model).

## appendChild()

El método **appendChild()** permite agregar un nuevo nodo hijo al final de la lista de nodos hijos de un elemento.

```jsx
const contenedor = document.getElementById("mi-contenedor");
const nuevoParrafo = document.createElement("p");
nuevoParrafo.textContent = "Nuevo párrafo";
contenedor.appendChild(nuevoParrafo);
```

## removeChild()

El método **removeChild()** se utiliza para eliminar un nodo hijo específico de un elemento.

```jsx
const contenedor = document.getElementById("mi-contenedor");
const primerParrafo = contenedor.firstChild;
contenedor.removeChild(primerParrafo);
```

## replaceChild()

El método **replaceChild()** permite reemplazar un nodo hijo existente por otro nuevo.

```jsx
const contenedor = document.getElementById("mi-contenedor");
const nuevoParrafo = document.createElement("p");
nuevoParrafo.textContent = "Texto reemplazado";
const primerParrafo = contenedor.firstChild;
contenedor.replaceChild(nuevoParrafo, primerParrafo);
```

## insertBefore()

El método **insertBefore()** permite insertar un nuevo nodo hijo antes de un nodo de referencia.

```jsx
const contenedor = document.getElementById("mi-contenedor");
const nuevoParrafo = document.createElement("p");
nuevoParrafo.textContent = "Nuevo párrafo";
const referencia = contenedor.firstChild;
contenedor.insertBefore(nuevoParrafo, referencia);
```

## cloneNode()

El método **cloneNode()** crea una copia del nodo actual, ya sea con o sin sus nodos hijos.

```jsx
const original = document.getElementById("original");
const copia = original.cloneNode(true); // Clonar con hijos
```

## Propiedades de Padres en JavaScript

## Introducción a las Propiedades de Padres

Las propiedades que te presentaré a continuación son útiles para acceder y trabajar con los elementos padres de un nodo en el DOM (Document Object Model).

### parentNode

La propiedad **parentNode** devuelve el nodo padre del elemento actual.

```jsx
const miElemento = document.getElementById("mi-elemento");
const padre = miElemento.parentNode;
```

### parentElement

La propiedad **parentElement** es similar a **parentNode**, pero devuelve el elemento padre directo.

## Nodos y Propiedad nodeName en JavaScript

## Introducción a los Nodos y nodeName

En el DOM (Document Object Model), los nodos son los elementos fundamentales que componen la estructura de un documento HTML. Cada nodo tiene un tipo, y para acceder a este tipo, puedes utilizar la propiedad **`nodeName`**.

### Tipos de Nodos

> Los tipos de nodos comunes incluyen:

- **Element Node (Nodo de Elemento):** Representa un elemento HTML, como <p></p> o <div></div>.
- **Text Node (Nodo de Texto):** Contiene el contenido de texto dentro de un elemento, como "Hola".
- **Comment Node (Nodo de Comentario):** Representa comentarios dentro del código HTML.
- **Document Node (Nodo de Documento):** Representa todo el documento HTML.

## nodeName

La propiedad **nodeName** devuelve el nombre del tipo de nodo en mayúsculas.

```jsx
const miElemento = document.getElementById("mi-elemento");
console.log(miElemento.nodeName); // Devuelve el nombre del tipo de nodo
```
